# trade-tracker-web
1.我有许多账户需要出钱，数额不一，我需要在这个网站上记录每一个出钱账户是什么，出多少钱，以及记录出掉一部分钱之后剩余待出钱的总量。
2.网站有一个账户编码，账户名称的数据库，输入账户的数字编码，网站自动能够输出账户名称
3.每一个账户占据一行，当我出给A交易对手钱后，我会在选定的出钱账户所对应的这一行中输入已出掉的金额，此时这行会新增一列‘已出金额n‘（n由新增交易次数决定），其值为出掉的金额，同时这一行的最后一列为扣减所有已出数额后此账户剩余的待出金额
4. 首次登录网站，会跳出一个录入本人姓名的弹窗，也可以选择进入浏览者模式；
4.1 如果录入本人姓名，此后会在页面中提示：您好，xxx；如果选择浏览者模式则提示：您已进入浏览模式
4.2 录入姓名情况下，在新页面中可以通过录入账户编码，此时会在账户名称栏位自动跳出对应账户名以供和对，再在交易金额栏位填入待出金额总量，点击添加账户后，会自动生成一行，管理人为姓名，的待出账记录
4.3 在已有待出账记录的行可以选择添加一笔出钱记录，这个添加可以一直进行到账户没有待出金额为止
5.正常情况下页面展示所有管理人的所有待出金额的记录
6.每个人可以勾选自己的账户，勾选后这一行变为无法更改状态，直接锁定，也可以点击解锁
7.每一行记录的最右侧添加一个删除键，点击后跳出确认删除提示，如果确认，则该行消失
8.每一行记录的‘已出金额n‘列可以通过点击其下方单元格，此时将自动复制一句标准版文字，‘账户名称 发 管理人 已出金额‘，使用Ctrl+V可以复制到聊天框里。举例“泰康资产XX年金 发王雅雯 5000w”
9.会话有效期管理：设置24小时会话有效期
10.过期会话清理：自动清理过期会话和Cookie
11.退出登录功能：添加退出登录按钮
12.浏览者模式下：
可以查看所有账户数据
可以复制支付信息文本
可以退出登录
不能添加新账户
不能添加支付记录
不能锁定/解锁账户
不能删除账户（按钮被禁用）
13.总金额可以修改
14.增加按钮下方对齐
15.一阶段：用户看不到彼此的记录，只能看到自己发布的
16.二阶段：发布时能选择项目是“公开/私有”
